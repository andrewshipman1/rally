Rally — the full theme system, animated.

11 themes from your PRD, each as a palette inside the same chassis. Same fonts. Same outlines. Same drop shadows. Same sticker treatment. Same motion language. Only the colors flex. Plus: the chassis is now alive — wiggling stickers, bouncing FOMO flags, marquee strips, entrance animations, pulsing shadows, blinking live-dots, hover wobbles. Refresh the page to see the entrance animations replay.

What's animated (this is the new motion kernel)

Always-on: the marquee at the top scrolls forever · the wordmark wiggles · the corner sticker wiggles · the FOMO flag bounces · the live-dot blinks · the countdown drop shadow pulses.

On load: sticker pops in with rotation · title slides up + bounces · countdown card slides in · avatars pop in one at a time in sequence.

On hover: the "I'm in" button does a quick wobble. (More hover micro-interactions in Phase 2.5.)

Theme audit (decisions for you)

Cut "Minimal." Directly contradicts the brand. It's on the is not list. Removing it.

"Couples Trip" reframed. Your protagonist is the friend group, not the couple. Below it's designed as a group of couples doing a vineyard weekend, not a romantic getaway for two. If you actually wanted a 2-person experience, that's a different product and probably doesn't belong in Rally — let me know.

What to look for:

All 11 themes share identical chassis behavior — same fonts, same outline weight, same sticker placement, same animation timing. The only thing that changes between any two themes is a 5-color CSS variable set. That's the entire flex. A future you (or a designer) can ship a new theme in ten minutes.

If any individual theme feels off, the fix is almost always one color choice — push the accent harder, dial the background warmer, swap the sticker fill. Don't change the chassis.
