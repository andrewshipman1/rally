9C — what to change

1 · Marquee — sell-specific dynamic content

  • Current: sketch theme content baked in ("★ usual suspects ★ throwback weekend ★ ..."). On sell, drive content from trip state.
  • Template: "★ {organizer.display_name} called you up ★ lock it in by {cutoffShort} ★ {inCount} already in ★".
  • cutoffShort = short date format (e.g., "mar 15"). Use date-fns.
  • If inCount === 0: omit the count segment. Don't render "0 already in".
  • All three segments separated by yellow stars (--accent).

2 · Marquee — scroll animation (CSS-only)

  • Infinite horizontal scroll via @keyframes. Duplicate the content track for seamless loop.
  • Animation duration 25–35s (tune for readability).
  • @media (prefers-reduced-motion: reduce) → pause animation. Content stays in place, legible.
  • No JS. No Intersection Observer. Pure CSS.

3 · Hero eyebrow

  • New small italic eyebrow above the trip title.
  • Format: "sell · {nights}-night trip" (e.g., "sell · 6-night trip"). If dates aren't set: "sell".
  • Style: Georgia italic, 11px, var(--muted).
  • Copy via getCopy + lexicon. Escalate if surface needs new entry.

4 · Hero live-dot row

  • New row below eyebrow: pulsing green dot + text.
  • Copy (sell, signed-in pre-RSVP): "trip is live".
  • Dot: 7px green (--live), 1.8s ease-in-out pulse loop, frozen on reduced-motion.
  • Style: 11px, muted color, bold, slight letter-spacing.

5 · Called-up sticker content

  • Current live: "still doing it 🥂" (or similar).
  • Sell (view 2+): "{daysToTrip} {themeEmoji}" format (e.g., "51 days 🌴").
  • themeEmoji pulls from theme tokens (palm for tropical, bride for bachelorette, etc.).
  • Escalation trigger: if sticker is a shared component used in multiple phases / views (teaser, view 2, organizer), flag options before overwriting. Don't silently break existing behavior.

6 · Trip title lowercase

  • Apply text-transform: lowercase to the hero title class.
  • Data stays as typed. Organizer-entered "Coachella 2026!!!" displays as "coachella 2026!!!".
  • CSS-only. Do NOT mutate trip.name. Do NOT lowercase at input time.
  • Font stays Georgia italic 900.

7 · Date + destination meta row

  • Below title: "{date_start_short} → {date_end_short} · {destination}" (e.g., "nov 20 → 26 · tortola, bvi").
  • 13px, var(--muted), 12px bottom margin.
  • Hide the row entirely if dates OR destination are unset.

8 · Theme postcard block

  • Horizontal rectangle below the meta row.
  • Theme-colored gradient (pull theme tokens — don't hardcode colors).
  • 2.5px ink border, 12px radius, 90px tall.
  • Destination stamp pill in top-right corner (white bg, 1.5px ink border, Georgia italic, slight rotation).
  • v0: static gradient — no real image, no Unsplash.
  • Hide the block if trip.destination is unset.
Escalation triggers to flag (per item 5 above): sticker component sharing, new lexicon entries needed for eyebrow / live-dot / marquee strings, and theme emoji source (theme token, theme-strings file, etc.).
