design + scope notes

phase gating

  • Sketch + Sell: playlist (the aux) only · packing/rules/album hidden.
  • Lock / Go / Done: all four extras render (playlist + packing + rules + album). Regression gate.
  • Gate at the render site via a phase prop on ExtrasSections. Don't delete any component code.

OG enrichment (the headline feature)

  • On save, POST URL to existing /api/enrich (same pipeline as lodging / headliner).
  • Store returned og_image + og_title on the playlist record.
  • Saved hero card renders with cover art as a left tile or background, title as primary text, domain chip in meta.
  • If enrich fails or returns empty: fall back to domain chip + generic ♫ icon. URL still saves.
  • Migration question: does the trips table have playlist_og_image/playlist_og_title columns? If not, Claude Code must flag before adding — escalation trigger.

equalizer bars

  • Four vertical bars, 3px wide, animated via @keyframes eq-bounce with staggered delays.
  • 1.2s ease-in-out infinite loop. Muted ink color.
  • Placed in the section header next to the title, before the caption.
  • Disabled under prefers-reduced-motion: reduce.

voice · playful

  • Title: "the aux" (not "the playlist")
  • Empty caption: "who's on?"
  • Saved caption: "aux cord secured"
  • Placeholder: "drop the link · spotify or apple music"
  • Hype hint (empty): "real fun starts when the crew piles on"
  • Open hint (saved): "tap the card to open · add songs from anywhere"
  • Byline: "set by {firstName} · {relativeTime}"
  • Swap affordance: "swap it" (abbreviated "swap" on fallback due to space)

interaction

  • Saved hero card: entire card is the anchor (<a target="_blank" rel="noopener">). Subtle hover lift.
  • "Swap it" pill: stops propagation, returns to input state.
  • No per-crew contributions, no embedded players, no song previews, no song counts — deferred.

out of scope

  • Repositioning the extras block on the trip page.
  • Redesigning packing / rules / album (they just get phase-gated — lock/go/done keep their current shape this session).
  • Per-crew song submissions or embedded players.
  • Color-extraction from album art (defer to future polish).
